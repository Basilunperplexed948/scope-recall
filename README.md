## 💡 How this tool works

The application functions as a background companion for your AI agent. It organizes memory in two distinct lanes. The first lane uses SQLite. This stores your project facts in a row and column format, much like a spreadsheet. This ensures the AI recalls exact dates, names, and requirements without errors. 

The second lane uses LanceDB. This creates a map of your ideas. It understands the meaning behind your text rather than just matching keywords. If you ask a question about a project, the tool gathers the right context from both sources. This hybrid approach gives the AI a complete picture of your work.

## 🔒 Understanding scope isolation

Scope isolation keeps your data tidy. Imagine a separate folder for every project you perform. The application ensures that data from Project A never mixes with Project B. This prevents the AI from becoming confused by unrelated information. You select the active scope inside the application menu. The AI then limits its memory to that specific boundary.

## ❓ Frequently asked questions

**Does this software send my data to the cloud?**
No. This tool runs locally on your Windows machine. Your data stays on your hard drive unless you choose to export it.

**Can I move my memory files to another computer?**
Yes. You can copy the folder containing your memory files to a new machine. Point the application to the new path in the settings menu.

**What happens if I delete a scope?**
Deleting a scope removes all memory associated with that project. Use caution as this action is permanent.

**Does the AI learn from my data?**
The tool acts as a storage provider for the AI. It does not train external models on your private data.
